# 2026年、AWS LambdaはVSCodeで開発するのがナウい

AWS Lambda開発の新潮流として、マネジメントコンソールに登場した「Open in Visual Studio Code」ボタンに着目。
この記事では、VS CodeとAWS Toolkit for VS Codeを連携させ、現代的かつ効率的なLambda開発フローを実践する方法を解説します。
クラウドとローカル環境をシームレスに繋ぎ、開発体験を向上させるための具体的な手順が紹介されています。
